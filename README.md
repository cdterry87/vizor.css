# Vizor.css
A CSS framework

## Dependencies
Live Server (Globally installed)

```
npm install -g live-server
```

## Get started
```
git clone https://github.com/cdterry87/vizor.css.git
npm install
```

## How to run (development) / build (production)
```
npm run start  # Run this while in development
npm run build  # Run this to build for production
```
